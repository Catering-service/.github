# Catering Service

Microservices-based catering platform built using Spring Boot and Spring Cloud.

## Architecture
- [Frontend App](https://github.com/Catering-service/frontend)
- [API Gateway](https://github.com/Catering-service/gateway-service) (Spring Cloud Gateway)
- [Service Discovery](https://github.com/Catering-service/eureka-server) (Eureka)
- [Auth Service](https://github.com/Catering-service/auth-service)
- Microservices
  - [User & Event Management Service](https://github.com/Catering-service/user-event-service)
  - [Finance Service](https://github.com/Catering-service/finance-service)
  - [Logistics & Partner Service](https://github.com/Catering-service/logistics-partner-service)
  - [Analytics & Admin Service](https://github.com/Catering-service/analytics-admin-service)

## Tech Stack

### Frontend
- React
- Vite

### Backend
- Java 21
- Spring Boot 4.0.x
- Spring Cloud
- Maven

### Data
- PostgreSQL

### DevOps
- Docker
